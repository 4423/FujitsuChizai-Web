# 学校案内アプリNAVI（サーバサイド）
![](https://4423lab.wpblog.jp/wp-content/uploads/2019/05/HmupbatQwLik8o.jpg)

サーバサイドは以下の機能を持つ。
+ 経路を提供するWebAPI
+ 地図を管理するWebページ
+ 経路案内の統計を表示するWebページ

![](https://4423lab.wpblog.jp/wp-content/uploads/2019/06/img1.png)
